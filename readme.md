## pie

Makes an array of pies.

## Getting Started
```

$git clone https://github.com/nss-evening-cohort-9/pie/
```
### Prerequisites
```
$npm install -g http-server
```

## Running
Browse to 'pie' directory and run http server.
```
$hs
```
